## [1.0.0-rc.1] — 2026-04-02 — Tour Data & API Fixes ✅

### Fixed
- **API Route Resend Crash**
  - Issue: `const resend = new Resend(process.env.RESEND_API_KEY)` at module level
  - Cause: Next.js attempts to collect route data at build time; missing env var causes crash
  - Solution: Moved Resend instantiation inside `POST` handler (lazy init)
  - Commit: `fix: move Resend init inside handler to prevent build-time crash`

### Updated
- **Tour Data: Beijing & Shanghai (`beijing-shanghai`)**
  - Replaced outdated itinerary with client-provided document (Apr 2)
