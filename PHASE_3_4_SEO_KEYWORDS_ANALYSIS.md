## Part 3: Tools 页面关键词机会

### Tool A: China Trip Cost Calculator
- **关键词:** "China trip cost calculator", "Travel cost China"
- **搜索量:** ~150-250/月
- **建议:** 交互式计算器，基于天数、城市数、住宿级别

### Tool B: Best Time to Visit (Weather + Crowds)
- **关键词:** "Best time to visit China", "China weather guide"
- **搜索量:** ~400-600/月 (高搜索量!)
- **建议:** 月度天气、降雨、人群、节假日日历

### Tool C: Visa Check & Requirements
- **关键词:** "China visa requirements New Zealand", "Visa checker China"
- **搜索量:** ~180-250/月
- **建议:** 国籍-目的-停留期限交互式检查器

### Tool D: Itinerary Builder / Recommender
- **关键词:** "China itinerary builder", "AI trip planner China"
- **搜索量:** ~100-180/月
- **建议:** 基于兴趣、时间、预算的自动行程推荐

### Tool E: Distance & Travel Time Calculator
- **关键词:** "China cities distance", "How long between cities"
- **搜索量:** ~120-200/月
- **建议:** 城市对搜索，交通方式比较
