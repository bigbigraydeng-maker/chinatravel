## 🚀 即刻可用步骤

### 步骤 1：环境配置（5 分钟）

```bash
# 编辑 .env.local
ITINERARY_ADMIN_PASSWORD=YourSecurePassword123

# 启动开发服务器
npm run dev
```

### 步骤 2：本地测试（10 分钟）

```
访问：http://localhost:3010/admin/itinerary-generator?key=YourSecurePassword123
填写表单 → 生成行程 → 预览 → 测试导出
```

### 步骤 3：部署到 Render（15 分钟）

```bash
# GitHub 推送
git add .
git commit -m "feat: add OP itinerary generator system"
git push origin main

# Render 自动部署
# 在 Dashboard 设置环境变量后自动完成
```

### 步骤 4：配置 Resend（20 分钟）

```
1. 访问 https://resend.com
2. 获取 API Key
3. 验证邮件域名
4. 在 Render 添加 RESEND_API_KEY
```

**总耗时：约 50 分钟**
